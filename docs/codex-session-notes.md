# Codex 에이전트 세션 기록

이 문서는 Codex 에이전트와 함께 진행한 세션을 간략히 메모하기 위한 템플릿입니다. 다음 항목을 참고해 필요한 내용을 빠르게 정리하세요.

## 체크리스트
- [ ] 세션 목적과 관련 Linear 이슈 ID
- [ ] 논의된 실험 아이디어 및 결정 사항
- [ ] 실행/테스트 명령어
- [ ] 후속 작업 담당자와 마감 일정

## 예시 양식
```
- 이슈: ACA-6 (코덱스 테스트)
- 주요 논의: 테스트용 저장소 상태 확인, 문서 정리 필요성 논의
- 실행/테스트: 별도 실행 없음
- 후속 작업: 필요 시 프로토타입 추가, 담당자 @username, 기한 yyyy-mm-dd
```

필요한 경우 추가 섹션을 자유롭게 확장해도 됩니다. 작은 메모라도 남겨 두면 이후 회고나 후속 작업에 도움이 됩니다.
