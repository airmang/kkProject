# 서비스 기능 범위 및 MVP 정의

## 1) 필수(MVP) vs 향후 확장 기능
- **필수(MVP)**
  - 온보딩 및 약관 동의
  - 위험 성향 및 목표 설문
  - 자산 규모/보유 자산군 입력
  - 목표 설정(기간, 목표 수익/손실 구간)
  - 자산배분 비율 제안(위험자산/헤지자산)
  - 리밸런싱 트리거 설정 및 알림(수익률·손실·주기)
  - 포트폴리오 요약 및 상태 대시보드
  - 로그/히스토리(설정 변경, 알림 기록)
  - 기본 보안/계정 관리(이메일·소셜 로그인, 비밀번호 변경)

- **향후 확장**
  - 다목표 관리(주택/교육/은퇴 등) 및 목표별 포트폴리오 분리
  - 세부 자산군 추천 템플릿(ETF/채권/금/현금 등 구체화)
  - 자동 리밸런싱 실행(P2P 연동 또는 브로커 API 연동)
  - 세금 시뮬레이션 및 절세 가이드
  - 공동계정/가족 포트폴리오 뷰
  - 커뮤니티/랭킹, 코치 챗봇 등 참여형 기능
  - 리스크 이벤트 모니터링(환율·금리·VIX 등)과 대응 제안
  - 백테스트/시나리오 시뮬레이터
  - 오프라인 자산(부동산 등) 입력 및 통합 자산 리포트

## 2) MVP User Stories
- 나는 **위험 성향을 알고 싶은 신규 사용자**로서, **과도한 손실을 피할** 수 있도록 **간단한 설문으로 내 위험 점수를 알고 싶다.**
- 나는 **자산배분 초보 투자자**로서, **목표에 맞는 위험/헤지 비율을 설정**하기 위해 **내 자산 규모와 목표 수익·손실 구간을 입력하고 추천 비율을 받고 싶다.**
- 나는 **시장 변동성이 걱정되는 사용자**로서, **사전에 정의한 규칙에 따라 대응**하기 위해 **수익률·손실 구간 또는 일정 주기를 기준으로 리밸런싱 알림을 받고 싶다.**
- 나는 **시간이 부족한 직장인**으로서, **현재 포트폴리오 상태를 빠르게 파악**하기 위해 **요약 대시보드와 최근 알림/조정 이력 로그를 확인하고 싶다.**

## 3) MVP 제외(구조만 선설계) 기능
- 다중 목표 관리와 목표별 자산배분 분리(데이터 모델에서 목표 단위 확장 고려)
- 자동 매매/리밸런싱 실행(외부 브로커 API 연동을 대비한 권한·거래내역 스키마 설계)
- 세금/환율 등 외부 데이터 연동(데이터 소스 확장성과 캐싱 전략 사전 검토)
- 가족·공동계정 및 공유 뷰(역할/권한 모델 초안 준비)
- 백테스트 및 시나리오 시뮬레이터(포트폴리오 히스토리와 가정 저장 구조 설계)
